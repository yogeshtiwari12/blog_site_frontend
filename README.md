Overview
This project is a feature-rich Blog Website designed and developed by Yogesh Tiwari. It offers users the ability to create, read, update, and delete blog posts while providing admins with additional controls to moderate and manage the platform. The website is built with the MERN stack to ensure seamless performance and scalability.

Features
User Features


Authentication: Secure sign-up and login using JWT-based authentication.

Profile Management: Users can create and customize their profiles.
Create and Edit Blogs: Authenticated users can write and manage their blog posts.
Comment System: Users can comment on blog posts and engage with other authors.
Search and Filter: Advanced search and filtering options for exploring blogs by title, category, or date.

Admin Features
Dashboard: View and manage all user accounts and blogs.
Approval System: Moderate content by approving or rejecting submitted blogs.
Manage Comments: Ability to remove or review inappropriate comments.

Additional Highlights
Responsive Design: Optimized for both desktop and mobile devices using Tailwind CSS.
Rich Text Editor: Provides a user-friendly experience for writing blogs.

Tech Stack

Frontend
React.js: For building the dynamic user interface.
Tailwind CSS: For styling and responsiveness.
Axios: For API requests.

Backend
Node.js: Server-side logic.
Express.js: Framework for building REST APIs.

Database
MongoDB: Database for storing user and blog data.
Installation and Setup
Prerequisites
Node.js installed
MongoDB set up locally or using a cloud provider (e.g., MongoDB Atlas)
